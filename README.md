# База данных Автосервис

Курсовой проект по проектированию и реализации базы данных для сети автомастерских на PostgreSQL.

## Техническая реализация

В проекте продемонстрировано использование продвинутых возможностей SQL и PL/pgSQL:

### Проектирование и нормализация
- Разработана ER-диаграмма предметной области
- Проведена нормализация отношений до Нормальной формы Бойса-Кодда (BCNF)
- Настроены индексы и ограничения целостности (Constraints)

### Сложные SQL запросы
- Использование CTE (Common Table Expressions) и рекурсивных запросов
- Оконные функции (Window Functions) для аналитических выборок
- Коррелированные подзапросы и сложные объединения (UNION, INTERSECT)
- Многоуровневые группировки и агрегатные функции

### Серверная логика
- Хранимые процедуры и функции на PL/pgSQL для инкапсуляции бизнес-логики
- Триггеры (Triggers) для аудита изменений и валидации данных
- Планирование фоновых задач через cron-подобные механизмы
- Использование курсоров для обработки наборов строк

### Управление транзакциями
- Работа с транзакциями (COMMIT, ROLLBACK)
- Демонстрация различных уровней изоляции транзакций

## Структура репозитория

Основные материалы проекта находятся в директории s1:

- new_create_tables.sql - DDL скрипты создания схемы базы данных
- new_inserts.sql - DML скрипты для начального заполнения таблиц
- er-diagramm-BKNF.png - ER-диаграмма базы данных в нотации BCNF
- cte_union_window.md - Примеры CTE и оконных функций
- correlirovannie_query.md - Реализация коррелированных подзапросов
- fn_proc.md - Исходный код хранимых процедур и функций
- triggersDima.md - Реализация триггерной логики
- transaction.md - Описание работы с транзакциями
